# Coding Test Solutions

This repository contains solutions for the coding test problems, written in **C++**.

## Files

- `program-1.cpp` – Small calculator using a class (add, subtract, multiply, divide).
- `program-2.cpp` – Generates the number series as per Problem‑2.
- `program-3.cpp` – Generates the odd-number pattern as per Problem‑3.
- `program-4.cpp` – Counts how many list elements are multiples of 1..9.
